Troubleshooting Tips:

  • File Not Found: Make sure the CSV file exists in the data/ folder
  • Invalid Data: Check that your CSV has the required columns: Date, Open, High, Low, Close
  • Empty File: Ensure your CSV file contains actual data rows
  • Date Format: Dates should be in a recognizable format (YYYY-MM-DD recommended)
  • Minimum Data: You need at least 30 rows of data for analysis
